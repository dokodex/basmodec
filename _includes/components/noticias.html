Como preparar una habitación antes de pintar

1. Proteger el espacio para evitar manchar. Para eso lo que necesitamos es:

- Cinta de carrocero para proteger interruptores, enchufes, zócalos…
- Plástico para cubrir muebles y cosas que queramos proteger.
- Papel o papel con cinta para cubrir el suelo u otras zonas.


2. Prepara la superficie a pintar. Rellenaremos las grietas que pueda haber en las paredes o techos con masilla extendiéndola con una espátula.

Debemos dejar secar la masilla y pasar una lija para que quede la superficie quede lisa y lista para pintar. Después pasamos un paño para quitar el polvo que haya quedado y ¡ya podemos comenzar a pintar!
